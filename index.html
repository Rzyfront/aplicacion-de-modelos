Presentación de la Actividad

Esta actividad académica tiene como objetivo analizar la aplicación de modelos de calidad en una página web específica. Seleccionaremos una red social o sitio web y aplicaremos los modelos de calidad McCall, Boehm y FURPS utilizando plantillas de evaluación con cuestionarios y escalas valorativas.

Descripción de los Modelos Utilizados

Modelo McCall

El modelo de calidad de McCall se centra en factores de calidad que afectan la operación, revisión y transición del software. Incluye factores como:

  • Correctitud
  • Fiabilidad
  • Eficiencia
  • Integridad
  • Usabilidad
  • Mantenibilidad
  • Flexibilidad
  • Portabilidad
  • Reusabilidad
  • Interoperabilidad

Modelo Boehm

El modelo de calidad de Boehm enfatiza la utilidad del software y su capacidad para ser mantenido y portado. Los factores principales son:

  • Utilidad
  • Mantenibilidad
  • Portabilidad

Modelo FURPS

FURPS es un acrónimo que representa los aspectos clave de calidad: Funcionalidad, Usabilidad, Fiabilidad, Rendimiento y Soporte. Es útil para evaluar tanto requisitos funcionales como no funcionales.

  • Funcionalidad (Functionality): Capacidad del sistema para realizar las funciones requeridas
  • Usabilidad (Usability): Facilidad de uso del sistema
  • Fiabilidad (Reliability): Capacidad para mantener su rendimiento bajo condiciones normales
  • Rendimiento (Performance): Eficiencia en el uso de recursos
  • Soporte (Supportability): Facilidad para ser modificado, probado y mantenido

Aplicación de las Plantillas de Evaluación

Aquí se aplicarán las plantillas de evaluación para cada modelo. Utilizaremos una escala valorativa del 1 al 5, donde:

  • 1 = Muy deficiente
  • 2 = Deficiente
  • 3 = Aceptable
  • 4 = Bueno
  • 5 = Excelente

¿El sitio web produce resultados correctos y precisos?

Fiabilidad

¿El sitio web es confiable y consistente en su funcionamiento?

Eficiencia

¿El sitio web utiliza recursos de manera eficiente?

Usabilidad

¿El sitio web es fácil de usar y entender?

Mantenibilidad

¿El sitio web es fácil de mantener y actualizar?

Mantenibilidad

Portabilidad
